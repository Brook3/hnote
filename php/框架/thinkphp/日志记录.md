# 1. 日志记录
开启/关闭自动日志记录
```php
# config.php
    // +----------------------------------------------------------------------
    // | 日志设置
    // +----------------------------------------------------------------------

    'log' => [
        // 日志记录方式，内置 file socket 支持扩展
        'type' => 'File',
        // 日志保存目录
        'path' => LOG_PATH,
        // 日志记录级别
        'level' => [],
    ],
```
关闭：可将 File 改为 test，或者日志记录级别改一下
