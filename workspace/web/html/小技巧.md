# 1. 小技巧
## 1.1. 跳转
> 写链接时可统一为 //brook3.com

如访问 127.0.0.1 可写为 //127.0.0.1 ，当属于https时会自动切换为https://127.0.0.1，如果是http则会切换为http://127.0.0.1
